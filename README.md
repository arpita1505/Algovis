## What is AlgoVis?

AlgoVis is an interactive web platform that visualizes how DSA algorithms work step by step. Every comparison, swap and traversal is animated in real time.

---

## Features

### Algorithm Visualizers
- **Sorting** — Bubble, Merge, Quick, Heap with pause/resume, step-by-step mode, speed control, custom array input, operation log and pseudocode panel
- **Graph** — BFS, DFS, Dijkstra with custom graph builder, add/remove nodes and edges, weighted edge editing
- **BST** — Insert, delete, search with animated traversal, balance indicator, inorder/preorder/postorder traversals
- **Search** — Binary vs Linear Search with live complexity comparison panel

### Advanced Features
- **Algorithm Race Mode** — Run two sorting algorithms on the same array simultaneously
- **Reference Dashboard** — Complexity bar charts, radar comparison, growth curves, algorithm deep-dives
- **75 DSA Problems** — Curated top interview problems (Easy to Hard) with problem statement, hints, full approach and LeetCode links. Progress saved locally.

### UI/UX
- Light and dark mode
- Smooth page transitions
- Mobile responsive navbar
- Custom input support on all visualizers
- Live status messages explaining each step
